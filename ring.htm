How to Add This Loading Animation to Your Website
  1. Copy ring.css to the root of your website or wherever you like.
    This CSS file is included with this document.
  2. Load the ring.css stylesheet into your HTML or CSS file.
    HTML: <link rel="stylesheet" href="yourwebsite.com/ring.css"/> CSS: @import url("yourwebsite.com/ring.css"); Replace "yourwebsite.com/ring.css" with the path where you place the ring.css file.
  3. Then, place this code wherever you want the animation to go. <span id="zrotate" class="rings"> <div id="vspin" class="ring1"> <div id="xspin" class="ring3"> <div id="zspin" class="ring2"> </div> </div> </div> </span>
